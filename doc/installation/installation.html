Spring++ Installation

You can install using the setup program or by downloading a ZIP package.

Installation using the setup program

Start spring++-0.1.0-win-x64-setup.exe, possibly set the containing directory and click the Install button. The 'containing directory' is a directory that will contain the installation directory. Setup creates a desktop icon and adds an environment variable SPRINGPP_ROOT that points to the installation directory.

Installation using a ZIP package

  • Extract the spring++-0.1.0.zip package to a directory of your choice.
  • Add an environment variable SPRINGPP_ROOT with a value that is a path to the directory you installed to.
  • The spring++ program is in the bin directory.
